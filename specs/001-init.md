# Harmonie Šternberk | Design Setup & Guidelines

This document outlines the visual identity and design system configuration for the Harmonie Šternberk symphonic wind orchestra digital presence.

## 1. Brand Identity & Vision
The Harmonie Šternberk brand bridges traditional Czech symphonic heritage with modern cinematic ambition. The visual language is **"Modern Classical"** — prestigious, spacious, and authoritative, yet accessible and dynamic.

## 2. Color Palette
The palette is built around the vibrant brand red (#e3051a), balanced by deep charcoals and warm parchment tones to maintain an elegant, high-culture feel.

| Role | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary (Brand Red)** | `#e3051a` | High-impact actions, brand logo, key accents, and active states. |
| **On-Primary** | `#ffffff` | Text/icons appearing on top of the brand red. |
| **Surface (Parchment)** | `#f9f9ff` | Primary background color, providing a clean "sheet music" feel. |
| **Surface Dim** | `#cfdaf1` | Subtle backgrounds, dividers, and secondary containers. |
| **On-Surface** | `#1a1c1e` | Main body text and secondary headings for high readability. |
| **Surface Container Low** | `#f0f3ff` | Subtle grouping elements and section backgrounds. |

## 3. Typography
Typography choices emphasize prestige and legibility, using a classic serif for personality and a clean sans-serif for functional UI.

- **Primary Heading Font:** `EB Garamond` (or similar high-contrast serif).
- **Secondary/UI Font:** Clean sans-serif (Inter/Roboto) for labels and metadata.
- **Hierarchy:**
    - **Headline Large:** 48px - 64px (Hero sections, cinematic titles)
    - **Headline Medium:** 32px - 40px (Section headings)
    - **Body Large:** 18px (Intro text, quotes)
    - **Body Medium:** 16px (Standard reading text)
    - **Label Small:** 12px - 14px (All-caps, wide tracking for dates/metadata)

## 4. Visual Elements & Shapes
- **Corner Radius:** `4px` (Minimal rounding to maintain a formal, structured look).
- **Separation:** Use thin `1px` borders in `primary/10` or generous whitespace rather than heavy shadows.
- **Imagery:** High-contrast photography, cinematic wide shots of the castle, and detailed close-ups of instruments.

## 5. Component Patterns
- **Navigation:** Transparent or surface-colored top bar with refined label-style links.
- **Buttons:**
    - **Primary:** Solid brand red (#e3051a) with white text.
    - **Secondary:** Outlined or charcoal-toned with subtle hover transitions.
- **Cards:** Flat, minimal elevation, using subtle background shifts (Surface Container Low) to define boundaries.

## 6. Implementation Notes
- **Contrast:** Ensure the brand red is used sparingly for emphasis to avoid overwhelming the classical aesthetic.
- **Spacing:** Use a 4px/8px grid system. Maintain generous "safe zones" around text to ensure the layout feels "breathable" like a musical score.
